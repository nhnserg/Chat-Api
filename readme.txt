# 💬 Real-Time WebSocket Chat Server

## 📌 Описание | Description

This is a WebSocket chat server built with Node.js, featuring JWT authentication, chat rooms, private messaging, and MongoDB message storage.

---

## 🚀 Возможности | Features

- ✅ JWT-аутентификация | JWT authentication
- 🗨️ Комнатный чат | Room-based chat
- 🔒 Приватные сообщения | Private messaging
- 🧾 Сохранение истории в MongoDB | Message history stored in MongoDB
- 🔔 Авто-доставка непрочитанных сообщений | Auto-delivery of unread messages
- ✍️ Индикация набора текста | Typing indicators
- 💓 Пинг/Понг и проверка активности | Ping/pong and client heartbeat

---
- **Node.js** + **ws**
- **MongoDB** + **mongoose**
- **JWT** for validation
- **sanitize-html** для защиты от XSS
