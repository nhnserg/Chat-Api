Используйте инструмент для тестирования WebSocket (например, wscat):

wscat -c ws://localhost:3000
Отправьте сообщение для подключения (замените TOKEN на реальный JWT токен):

{
  "type": "join",
  "roomId": "room_id",
  "token": "TOKEN"
}
Отправка сообщения в комнату:

{
  "type": "message",
  "roomId": "room_id",
  "content": "Привет!",
  "token": "TOKEN"
}
Отправка личного сообщения:

{
  "type": "privateMessage",
  "recipient": "username",
  "content": "Привет!",
  "token": "TOKEN"
}
Для полного тестирования рекомендуется открыть несколько подключений wscat и проверить:

Подключение к комнате
Отправку сообщений в комнату
Личные сообщения
Статус печати
Отключение пользователей
